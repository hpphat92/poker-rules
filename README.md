# Execute the program

run `node main.js`


## Input
None

## Output
The first line will be the name of the rule either of: 
- royalFlush
- straightFlush
- fourOfKind
- fullHouse
- flush
- straight
- threeOfKind
- twoPair
- onePair
- highCard

The second line will show the list of card with the rank and the suit

```
[
  Card { rank: '4', suit: 'heart' },
  Card { rank: '4', suit: 'diamond' }
]

```
